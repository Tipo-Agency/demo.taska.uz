# taska.uz - Система управления задачами

Демо-версия системы управления задачами для потенциальных клиентов.

🌐 **Демо:** https://demo.taska.uz

## 🚀 Быстрый старт

**Требования:** Node.js 18+

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите локально:
   ```bash
   npm run dev
   ```

3. Откройте в браузере: http://localhost:3000

## 📦 Сборка для продакшна

```bash
npm run build
```

Собранные файлы будут в папке `dist/`

## 🌐 Деплой

Подробные инструкции по деплою на `demo.taska.uz` смотрите в [DEPLOY.md](DEPLOY.md)

## ✨ Особенности

- ✅ Полностью локальное приложение (без backend)
- ✅ Все данные хранятся в localStorage
- ✅ Готовые мок-данные для демонстрации
- ✅ Адаптивный дизайн
- ✅ Темная/светлая тема
- ✅ Управление задачами, сделками, клиентами
- ✅ CRM, финансы, склад, бизнес-процессы

## 🎯 Для демонстрации

При первом запуске автоматически загружаются демо-данные:
- 4 пользователя (admin/admin для входа)
- Задачи, сделки, клиенты
- Документы, встречи, контент-план
- Финансовые данные и склад

## 📝 Технологии

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Lucide Icons
